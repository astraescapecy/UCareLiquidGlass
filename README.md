# Bloom (UCareLiquidGlass)

SwiftUI self-improvement app: multi-select growth areas → daily time envelope → synthesized routine playbook → checklist home screen. Liquid-glass materials, calm dusk/teal palette, spring motion with Reduce Motion fallbacks, persisted with `UserDefaults`.

## Xcode

From this directory:

```bash
xcodegen generate
open UCareLiquidGlass.xcodeproj
```

Simulator: build & run targeting **iPhone 17** (or any iOS 17+ Simulator).
